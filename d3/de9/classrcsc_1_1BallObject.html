Detailed Description

observed ball object class

order of update process. update() -> updateAll() (updatePos(),updateOnlyVel(), updateOnlyRelativePos()) -> updateByHeardInfo() -> updateByCollision() -> updateByGameMode() -> updateSelfRelated()
